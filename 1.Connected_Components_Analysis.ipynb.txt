    "## Step 2: Component Definition & Configuration\n",
    "\n",
    "Here we document what constitutes a **minimum component** and define configuration settings. A minimum component is defined as a set of connected pixels of non-background color with a size greater than or equal to `MIN_COMPONENT_SIZE`.\n",
    "\n",
    "**Documentation:**\n",
    "1. **Background**: In ARC, color `0` (black) is standardly treated as the background. Thus, any pixels with color `0` are excluded from component identification.\n",
    "2. **Same Color vs. Non-Background**:\n",
    "   - *Same Color*: Components are contiguous pixels sharing the *exact same* color value.\n",
    "   - *Non-Background*: Components are contiguous pixels of *any* non-background color (values > 0), potentially forming multi-colored composite objects.\n",
    "3. **Adjacency (Connectivity)**:\n",
    "   - *4-Adjacent*: Connected vertically and horizontally (orthogonal).\n",
    "   - *8-Adjacent*: Connected vertically, horizontally, and diagonally.\n",
    "4. **Minimum Component Size**: We set `MIN_COMPONENT_SIZE` to 1. Single pixels of non-background colors are considered valid components."
   ]

    "## Step 5: Connected Components Labeling & Feature Extraction Logic\n",
    "\n",
    "We now implement the core algorithm to segment connected components. We separate between orthogonal (4-connected) and orthogonal+diagonal (8-connected) adjacencies, and between Same Color grouping and Non-Background grouping. For each labeled component, we extract sizes, shapes, bounding boxes, solidity, and centroids."

   "cell_type": "code",
   "execution_count": null,
   "id": "73de9433",
   "metadata": {},

   "source": [
    "# Structuring elements\n",
    "STRUCT_4 = np.array([[0, 1, 0],\n",
    "                     [1, 1, 1],\n",
    "                     [0, 1, 0]])\n",
    "\n",
    "STRUCT_8 = np.array([[1, 1, 1],\n",
    "                     [1, 1, 1],\n",
    "                     [1, 1, 1]])\n",
    "\n",
    "def extract_components_from_grid(grid, connectivity, group_by_color):\n",
    "    \"\"\"\n",
    "    Extracts list of component dictionaries from a grid.\n",
    "    connectivity: 4 or 8\n",
    "    group_by_color: bool (True for same-color, False for non-background)\n",
    "    \"\"\"\n",
    "    struct = STRUCT_4 if connectivity == 4 else STRUCT_8\n",
    "    h, w = grid.shape\n",
    "    components = []\n",
    "    \n",
    "    if group_by_color:\n",
    "        # For same-color grouping, examine each unique color > 0\n",
    "        unique_colors = np.unique(grid)\n",
    "        unique_colors = unique_colors[unique_colors != CONFIG['BACKGROUND_COLOR']]\n",
    "        \n",
    "        for color in unique_colors:\n",
    "            mask = (grid == color)\n",
    "            labeled_arr, num_features = label(mask, structure=struct)\n",
    "            \n",
    "            for label_idx in range(1, num_features + 1):\n",
    "                comp_mask = (labeled_arr == label_idx)\n",
    "                size = np.sum(comp_mask)\n",
    "                if size < CONFIG['MIN_COMPONENT_SIZE']:\n",
    "                    continue\n",
    "                \n",
    "                # Bounding box & geometry\n",
    "                rows, cols = np.where(comp_mask)\n",
    "                min_row, max_row = int(np.min(rows)), int(np.max(rows))\n",
    "                min_col, max_col = int(np.min(cols)), int(np.max(cols))\n",
    "                box_h = max_row - min_row + 1\n",
    "                box_w = max_col - min_col + 1\n",
    "                \n",
    "                solidity = size / (box_h * box_w)\n",
    "                centroid_r = np.mean(rows)\n",
    "                centroid_c = np.mean(cols)\n",
    "                \n",
    "                # Touches border check\n",
    "                touches_border = bool(\n",
    "                    min_row == 0 or max_row == h - 1 or\n",
    "                    min_col == 0 or max_col == w - 1\n",
    "                )\n",
    "                \n",
    "                components.append({\n",
    "                    'color': int(color),\n",
    "                    'size': int(size),\n",
    "                    'box_h': box_h,\n",
    "                    'box_w': box_w,\n",
    "                    'aspect_ratio': box_w / box_h,\n",
    "                    'solidity': solidity,\n",
    "                    'centroid_row': centroid_r,\n",
    "                    'centroid_col': centroid_c,\n",
    "                    'normalized_centroid_row': centroid_r / h if h > 1 else 0.5,\n",
    "                    'normalized_centroid_col': centroid_c / w if w > 1 else 0.5,\n",
    "                    'touches_border': touches_border\n",
    "                })\n",
    "    else:\n",
    "        # For non-background grouping, label all elements > 0 together\n",
    "        mask = (grid != CONFIG['BACKGROUND_COLOR'])\n",
    "        labeled_arr, num_features = label(mask, structure=struct)\n",
    "        \n",
    "        for label_idx in range(1, num_features + 1):\n",
    "            comp_mask = (labeled_arr == label_idx)\n",
    "            size = np.sum(comp_mask)\n",
    "            if size < CONFIG['MIN_COMPONENT_SIZE']:\n",
    "                continue\n",
    "            \n",
    "            rows, cols = np.where(comp_mask)\n",
    "            min_row, max_row = int(np.min(rows)), int(np.max(rows))\n",
    "            min_col, max_col = int(np.min(cols)), int(np.max(cols))\n",
    "            box_h = max_row - min_row + 1\n",
    "            box_w = max_col - min_col + 1\n",
    "            \n",
    "            solidity = size / (box_h * box_w)\n",
    "            centroid_r = np.mean(rows)\n",
    "            centroid_c = np.mean(cols)\n",
    "            \n",
    "            touches_border = bool(\n",
    "                min_row == 0 or max_row == h - 1 or\n",
    "                min_col == 0 or max_col == w - 1\n",
    "            )\n",
    "            \n",
    "            # Determine dominant color\n",
    "            colors_in_comp = grid[comp_mask]\n",
    "            vals, counts = np.unique(colors_in_comp, return_counts=True)\n",
    "            dominant_color = int(vals[np.argmax(counts)])\n",
    "            \n",
    "            components.append({\n",
    "                'color': dominant_color,  # Dominant color inside\n",
    "                'size': int(size),\n",
    "                'box_h': box_h,\n",
    "                'box_w': box_w,\n",
    "                'aspect_ratio': box_w / box_h,\n",
    "                'solidity': solidity,\n",
    "                'centroid_row': centroid_r,\n",
    "                'centroid_col': centroid_c,\n",
    "                'normalized_centroid_row': centroid_r / h if h > 1 else 0.5,\n",
    "                'normalized_centroid_col': centroid_c / w if w > 1 else 0.5,\n",
    "                'touches_border': touches_border\n",
    "            })\n",
    "            \n",
    "    return components\n",
    "\n",
    "print('Labeling and logic compiled successfully!')"

    "# Hypothesis: 8-Adjacent Connectivity results in fewer but larger connected components than 4-Adjacent Connectivity.\n",
    "\n",
    "In this section, we test whether utilizing diagonal adjacency (8-adjacent) merges distinct orthogonal objects (4-adjacent) into fewer, larger spatial units."
   ]

    "### Methodology\n",
    "\n",
    "We compute and compare grid-level component statistics for both connectivity methods.\n",
    "1. **Paired Observations**: Each grid yields two pairs of measurements: (4-same vs 8-same) and (4-nonbg vs 8-nonbg).\n",
    "2. **Metrics**: We evaluate the number of components per grid, and the mean size of components per grid.\n",
    "3. **Statistical Test**: Since data distributions are non-normal and paired, we use the **Wilcoxon signed-rank test** to assess statistical significance. we also report median differences and effect sizes."
   ]
  },

    "### Hypotheses\n",
    "\n",
    "- **Null Hypothesis ($H_0$)**: There is no difference in the component count or the mean component size per grid between 4-adjacent and 8-adjacent configurations.\n",
    "- **Alternative Hypothesis ($H_1$)**: 8-adjacent configuration results in a statistically significant lower number of components and a larger mean component size per grid than 4-adjacent configuration."

    "### Methodology\n",
    "\n",
    "We analyze components extracted under the 8-adjacent same-color rule and the 8-adjacent non-background rule. We measure and compare the following individual component features:\n",
    "1. **Size**: Number of non-background pixels.\n",
    "2. **Solidity**: The compactness of the component (size divided by bounding box area). Higher solidity means a denser shape.\n",
    "3. **Aspect Ratio**: Width divided by height of the component's bounding box.\n",
    "4. **Statistical Test**: Since these are independent component populations of different sizes, we use the **Mann-Whitney U test** to compare distributions."
   ]

    "- **Alternative Hypothesis ($H_1$)**: Same-Color components exhibit significantly lower sizes, different solidity, and aspect ratio distributions compared to Non-Background components due to the separation of multi-colored objects into smaller mono-colored pieces."
